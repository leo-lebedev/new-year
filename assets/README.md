# Инструкция по добавлению музыки

Для работы фоновой музыки необходимо добавить музыкальный файл в эту папку.

## Как добавить музыку:

### Вариант 1: Использовать свою музыку
1. Найдите новогоднюю песню в формате MP3
2. Переименуйте файл в `music.mp3`
3. Поместите файл в папку `assets/`

### Вариант 2: Скачать бесплатную новогоднюю музыку

Вы можете найти бесплатную новогоднюю музыку на следующих сайтах:

1. **YouTube Audio Library** - https://studio.youtube.com/
   - Перейдите в "Библиотека аудио"
   - Найдите новогоднюю музыку
   - Скачайте и используйте бесплатно

2. **Free Music Archive** - https://freemusicarchive.org/
   - Поиск по тегу "Christmas" или "New Year"
   - Скачайте треки с лицензией Creative Commons

3. **Pixabay Music** - https://pixabay.com/music/
   - Бесплатная музыка без авторских прав
   - Поиск "Christmas music" или "Holiday music"

4. **Incompetech** - https://incompetech.com/
   - Раздел "Royalty Free Music"
   - Категория "Holiday"

### Вариант 3: Использовать онлайн URL

Если вы хотите использовать музыку по прямой ссылке:

1. Откройте файл `index.html`
2. Найдите строку:
   ```html
   <source src="assets/music.mp3" type="audio/mpeg">
   ```
3. Замените на:
   ```html
   <source src="ВАШ_URL_К_МУЗЫКЕ.mp3" type="audio/mpeg">
   ```

## Поддерживаемые форматы:
- MP3 (рекомендуется)
- OGG
- WAV

## Рекомендации:
- Используйте файлы размером не более 5-10 МБ для быстрой загрузки
- Оптимальное качество: 128-192 kbps
- Убедитесь, что у вас есть права на использование музыки

---

**Примечание:** После добавления файла `music.mp3` в эту папку, просто обновите страницу в браузере, и музыка начнет воспроизводиться автоматически (если браузер разрешает автовоспроизведение).
